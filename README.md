# DMGC
This is a Tensorflow implementation of the paper: Deep Multi-Graph Clustering via Attentive Cross-Graph Association

WSDM 2020

## Requirements

  * Python 3.6.2
  * tensorflow (>0.12)
  * networkx

## Usage
### train the model
	python run_exp.py
### show visualization results
	python vis.py
